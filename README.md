# 🔍 ShareDebug（分享调试器）

> 一键捕获 Android 应用分享数据的开发者工具

---

## 🎯 用途

开发 Android 应用时，测试分享功能是否正常工作，或分析其他应用分享的数据格式。

---

## ⚡ 功能

| 功能 | 说明 |
|------|------|
| **捕获分享** | 接收 `ACTION_SEND` / `ACTION_SEND_MULTIPLE` Intent |
| **完整解析** | 显示 Action、Type、URI、Extras 等全部数据 |
| **文本提取** | 展示分享的纯文本内容和主题 |
| **多文件支持** | 列出多文件分享时的所有 URI |
| **一键复制** | 将完整调试信息复制到剪贴板 |
| **实时刷新** | 支持多次分享，自动覆盖显示新数据 |

---

## 🚀 使用方法

1. **安装应用** —— 安装 ShareDebug APK
2. **发起分享** —— 在任意应用中点击"分享"
3. **选择调试器** —— 在分享列表中选择"ShareDebug"
4. **查看数据** —— 界面显示该次分享的完整 Intent 信息
5. **复制分析** —— 点击"复制"将数据粘贴到其他地方分析

---

## 💡 适用场景

- 调试自家应用的分享功能输出
- 分析第三方应用分享的数据结构
- 验证分享接收方是否正确解析数据
- 排查分享内容丢失或格式错误问题
